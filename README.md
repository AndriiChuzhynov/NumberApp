# NumbersApp

### Requirements
Installed Go version: go1.12.7 and higher  
If no Go installed, please follow instruction:
 https://golang.org/doc/install  
2+ CPU, 4GiB RAM  

### Supported platforms:
  - MacOS Mojave and higher
  - Ubuntu 18.04
  - RedHat 7+ 
  
### Run:
Type `go run .`  
Or `go build && ./NumberApp`  
Application will listen `4000` port
